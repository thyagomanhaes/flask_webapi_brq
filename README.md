# flask_webapi_brq
Resolução para # 2 Web Api

Para rodar o app:

1. Crie um ambiente com o virtualenv
>> virtualenv env

2. Instale as dependencias do projeto usando requirements.txt
>> pip install -r requirements.txt

3. Ative o ambiente
>> source env/Scripts/activate (para bash)

4. Rode o programa
>> python app.py
